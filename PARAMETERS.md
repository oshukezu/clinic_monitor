# 參數設定指南 (Configuration Guide)

本專案提供兩個主要設定檔，讓您可以調整監控目標與範圍。

## 1. 關鍵字與搜尋設定 (`seo_config.py`)

請編輯 `seo_config.py` 檔案來修改關鍵字。

```python
# 範例
KEYWORDS = [
    "中醫",    # 核心關鍵字
    "減重",    # 新增關鍵字
    "不孕症"   # 新增關鍵字
]
```

- **SEARCH_ZOOM**: 控制搜尋範圍半徑。預設為 `15z` (約 1 公里)。若需更廣範圍可改為 `14z`，更精確範圍可改為 `16z`。
- **CACHE_TTL**: 資料更新頻率。預設為 `604800` (7天)。若要改成每天更新，請設為 `86400`。

## 2. 診所名單與座標 (`clinics_config.py`)

請編輯 `clinics_config.py` 來新增或修改診所據點。

```python
CLINICS = {
    "診所名稱": {"lat": 24.123456, "lng": 120.123456},
    # ...
}
```

- **名稱**: 請填入您習慣稱呼的名稱（系統會自動模糊比對 Google Map 上的真實店名）。
- **lat/lng**: 該診所的經緯度座標。建議直接從 Google Maps 右鍵點選 "這是哪裡？" 取得精確數值。

## 注意事項

- **API 費用**: 每次完整執行會消耗 `診所數量 * 關鍵字數量` 次的 API 呼叫。例如 11 家診所監控 5 個關鍵字，一次消耗 55 點。請留意 SerpApi 的額度。
